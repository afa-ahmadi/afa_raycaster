## Running the Project

Please run the code using a terminal or WSL (Windows Subsystem for Linux).

```bash
make
make run
```

---

## Installing raylib (Linux / WSL)

### Ubuntu / Debian (recommended)

```bash
sudo apt update
sudo apt install libraylib-dev
```
